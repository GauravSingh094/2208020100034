# URL Shortener Project

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

Backend will run on http://localhost:5000

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Frontend will run on http://localhost:3000

## API Docs
- POST /api/shorten
- GET /s/:shortId
- GET /api/stats/:shortId
